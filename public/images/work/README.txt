Drop your real "Our Work" photos in this folder, then open
src/lib/site.ts and set the `src` for each item in the `ourWork` array, e.g.:

  { title: "Suits & formal wear", ..., src: "/images/work/suit-1.jpg" }

Recommended: landscape photos ~1200x900px (4:3), JPG or WebP, under ~400KB each.
Leaving src as null shows a styled "Photo soon" placeholder slot.
